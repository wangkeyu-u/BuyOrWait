# Product

## Register

product

## Users
Hackathon judges, PC gamers, and game developers evaluating AI-assisted game purchase decisions. Users need to understand whether a game is worth buying now, whether recent reviews contradict lifetime ratings, and whether negative review spikes may indicate review bombing.

## Product Purpose
BuyOrWait is a game purchase decision tool that turns recent weighted sentiment, trend signals, deal momentum, and anomaly detection into simple Buy / Wait / Skip recommendations. Success means the demo first feels like a premium 3D AI product website, runs with mock data, and can later swap in backend or BigQuery aggregates without changing the interface.

## Brand Personality
Sharp, playful, premium-intelligent. The product should feel like an Apple/Vision Pro-inspired 3D game decision assistant: energetic enough for a hackathon demo, but trustworthy enough for data-backed decisions.

## Anti-references
Avoid plain admin tables, generic SaaS landing pages, presentation-only mockups, dark hacker dashboards, Steam monitor-wall aesthetics, and any UI that hides the actual recommendation logic behind vague visual effects.

## Design Principles
Show the decision first, then the evidence.
Make recent sentiment feel more trustworthy than lifetime rating.
Use 3D game-like atmosphere without sacrificing dashboard readability.
Keep backend boundaries replaceable and secrets-free.
Let anomalies feel urgent, but not noisy.

## Accessibility & Inclusion
Use readable contrast, non-color labels for Buy / Wait / Skip and severity, keyboard-friendly native Streamlit controls, and reduced-motion fallbacks for CSS/JS effects where practical.
